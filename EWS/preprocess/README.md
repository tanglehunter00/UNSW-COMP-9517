# 文献综述（Literature Review）

## Classic Techniques

早期田间或冠层相关任务常依赖手工设计的颜色与植被指数、经验阈值分割，以及决策树、支持向量机、随机森林等传统机器学习分类器。这类方法在光照与土壤背景相对可控时尚可工作，但在室外复杂场景下往往需要反复调参、泛化有限。随着高通量表型与移动成像平台普及，学界逐步转向以卷积网络为代表的深度分割模型；经典方法多作为基线或与深度模型对照出现。

## Modern Techniques

### 1. 室外小麦植株分割与 EWS 数据集（DeepLab v3+）

Van Gool 等人在 *Frontiers in Plant Science* 发表的工作面向高通量田间表型，在瑞士 Eschikon 田间平台采集的多季冬小麦 RGB 影像上，构建 Eschikon Wheat Segmentation（EWS）像素级标注数据，并采用 DeepLab v3+（ResNet 编码器、ImageNet 预训练）进行植株与土壤二分类分割，指标上优于基于支持向量机与随机森林的传统流程。该文系统讨论了室外光照、土壤湿度与基因型差异带来的成像难点，是本项目下游分割数据与任务设定的重要直接来源。

全文链接：<https://www.frontiersin.org/journals/plant-science/articles/10.3389/fpls.2021.774068/full>

### 2. 三维医学图像的自回归序列预训练

Wang 等人的工作针对 CT、MRI 等三维医学影像，提出以自回归方式对「视觉 token」序列建模的预训练框架：依据空间、对比与语义关联将体素或局部块组织成序列，通过预测下一 token 学习上下文表征，并配合随机起点等策略缓解对 token 关系的过强假设。其思路体现了在缺乏大量标注时，通过生成式或自回归自监督任务挖掘长程依赖的现代范式；与掩码自编码器等方法同属自监督表征学习脉络，可为理解「无标签预训练再微调」提供参照，尽管任务域与二维田间 RGB 不同。

PDF：<https://arxiv.org/pdf/2409.08691>

### 3. DFormerv2：几何先验与自注意力（RGBD 语义分割）

Yin 等人提出 DFormerv2，在 RGBD 语义分割中强调深度可视为几何补充，进而从深度与 patch 间空间距离中提取几何线索，将其作为先验调节自注意力权重，而非简单用另一路网络「等同编码」深度图。在多个 RGBD 分割基准上表现突出。本项目在二维 MAE 编码器中使用的距离相关注意力偏置，在概念上与「利用 patch 空间关系塑造注意力」这一脉络一致；田间任务无深度通道时，需在纯 RGB patch 网格上近似类似几何先验。

PDF：<https://arxiv.org/pdf/2504.04701>

---

以上三篇与「预处理—无标签预训练—EWS 微调」流水线的关系可概括为：第一篇界定数据与监督分割基线方法；第二篇代表医学影像领域的自回归自监督预训练；第三篇为几何偏置注意力的代表性近期工作，与几何 MAE 设计动机相呼应。
